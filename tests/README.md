# Testing Suite

This folder contains the unit and integration tests for the Stacks Builder Challenge.

## Framework
- **Vitest**: Used for fast, reliable unit testing.
- **Clarinet**: Used for Clarity smart contract testing (in the `tests/` directory at the root).

## Running Tests
Run the frontend test suite with:
```bash
npm test
```
To run Clarity contract tests:
```bash
clarinet test
```
