# Styles & Design System

This folder contains the CSS modules and global styles that define the application's look and feel.

## Visual Philosophy
- **Modern & Premium**: Uses sleek gradients, glassmorphism effects, and micro-animations.
- **Responsive**: All components are designed with a mobile-first approach, ensuring a seamless experience across all device sizes.
- **Consistent**: Utilizes a central set of CSS variables for colors, spacing, and typography.

## Key Files
- `globals.css`: Global styles, CSS variables, and reset logic.
- `components.css`: Specific styles for reusable UI elements.
