# Stacks Developer Resources

A curated list of essential resources for developing on the Stacks blockchain with Clarity.

## 📚 Official Documentation
- **[Stacks Documentation](https://docs.hiro.so/)**: Comprehensive guides for the entire Stacks ecosystem.
- **[Clarity Language Reference](https://docs.stacks.co/docs/write-smart-contracts/clarity-language)**: Detailed reference for every Clarity function and keyword.
- **[SIPs (Stacks Improvement Proposals)](https://github.com/stacksgov/sips)**: The standards that define the Stacks network.

## 🛠 Developer Tools
- **[Clarinet](https://github.com/hirosystems/clarinet)**: Local development environment and testing framework.
- **[Hiro Wallet](https://wallet.hiro.so/)**: Browser extension and desktop wallet for interacting with apps.
- **[Stacks Explorer](https://explorer.hiro.so/)**: Tool for inspecting transactions, blocks, and contracts.

## 🎓 Learning Material
- **[Clarity Universe](https://clarity-lang.org/universe)**: A guided learning path for Clarity developers.
- **[Stacks Academy](https://stacks.academy/)**: Educational content for all levels of expertise.

## 💬 Community & Support
- **[Stacks Discord](https://discord.gg/stacks)**: Connect with other developers and the core team.
- **[Stacks Forum](https://forum.stacks.org/)**: Governance and technical discussions.
- **[Hiro Developer Twitter](https://twitter.com/hirosystems)**: Updates on tools and features.

## 📦 SDKs
- **[Stacks.js](https://github.com/hirosystems/stacks.js)**: The standard JavaScript/TypeScript library for Stacks.
